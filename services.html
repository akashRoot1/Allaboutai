What do I get with the free templates? +

What's included in the Pro Pack? +

How does the Done-For-You service work? +
After purchase, you'll receive a calendar link to book a 1-hour Zoom call. We'll set up your n8n instance (Railway or Docker), configure all credentials, import and test all workflows, tune the AI prompts to match your resume, and make sure everything is running live before the call ends. Do I need coding experience? +
Not for the free templates or Pro Pack — n8n is visual and the workflows come pre-built. You'll need to fill in your API credentials (step-by-step guides provided) but no coding is required. The Done-For-You service handles everything for non-technical users.
Is there a refund policy? +
Yes — 30-day money-back guarantee on the Pro Pack and Done-For-You service, no questions asked. If you're not satisfied, email hi@thatai.me within 30 days and we'll refund you in full.
What tools and subscriptions do I need? +
For the free tier: n8n (free cloud or self-hosted), OpenAI API (~$5/month for daily use), RapidAPI free tier, and a Google account. Monthly running cost is roughly $5-10. The Pro Pack and DFY service include the same cost structure — you pay for the APIs you use, we just automate them.
